# 🏷️ Coupon Redeemer Web App

A simple and responsive coupon redemption web application built to showcase available discount codes and allow users to redeem them easily. Deployed at: [https://coupons-qjwj.onrender.com](https://coupons-qjwj.onrender.com)

## 🔗 Live Demo
👉 [Visit Live Site](https://coupons-qjwj.onrender.com)

---



## 🚀 Features

- 🧾 List of available coupons with titles and descriptions
- 🎯 Direct link to redeem coupons
- 💻 Fully responsive design (mobile and desktop friendly)
- ⚡ Fast and lightweight deployment on Render

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, TailwindCSS (or your framework)
- **Backend (Optional):** Flask / Node.js / Static hosting (depending on your setup)
- **Hosting:** Render
